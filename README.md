# Solana Commerce App (skeleton)

This repository contains a minimal Node.js backend for a Solana-based commerce application. It demonstrates:

- User registration and login
- Depositing SOL into a simple in-memory balance
- Trading tokens (placeholder endpoint)
- Purchasing goods with automatic 0.20% commission
- Admin endpoint to list transactions

> **Note:** This is a skeleton implementation. Real Solana integration, authentication, database storage and fiat conversion are **not** implemented.

## Backend

```bash
cd backend
npm install
npm start
```

## Testing

```bash
npm test
```
